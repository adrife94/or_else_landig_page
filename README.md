# OR ELSE - Discipline or Regret

A hardcore productivity app landing page based on military discipline and negative reinforcement.

## Project Structure
- `index.html`: Main landing page structure
- `styles.css`: Dark luxury design system with #FF0000 accent
- `script.js`: Animations and interactions
- `images/`: Asset directory (currently generating...)

## Setup
Open `index.html` in your browser to view the page.
